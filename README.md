# AHU-DWNET-Client
![alt text](/Assets/logo.png)

安徽大学(AHU)校园网登录应用 使用winui3
##  功能
- 登录
- 登出
- 自动保存账号密码
- 自动登录
- 网络类型检测(不同网络请求方式不同)
